---
title: Output Submitter
---

The output submitter bot is respondsible for finalizing the minitia's block. It does this by receiving the block output from the bridge executor, and submitting it to the Initia L1 as output proposals.

## Set Output Submitter Environment Variables

<Note>
- `TYPEORM_DATABASE` should be the name of the database previously created
- `L2_CHAIN_ID` is your minitia's chain ID and bridge executor menmonic found in `$HOME/.minitia/artifacts/config.json`
- `BRIDGE_ID` can be found in `$HOME/.minitia/artifacts/artifacts.json`
- `OUTPUT_SUBMITTER_MNEMONIC` can be found in `$HOME/.minitia/artifacts/config.json`'s `ouput_submitter` field
</Note>

```sh .env.output
TYPEORM_CONNECTION=postgres               # database connection (currently only support `postgres`)
TYPEORM_HOST=localhost                    # database host
TYPEORM_USERNAME=postgres                 # database username
TYPEORM_PASSWORD=postgres                 # database password
TYPEORM_DATABASE=minitia                  # database name
TYPEORM_PORT=5432                         # database port
TYPEORM_SYNCHRONIZE=true                  # synchronize database schema
TYPEORM_LOGGING=false                     # enable logging
TYPEORM_ENTITIES=dist/orm/*Entity.js      # entity path

L1_CHAIN_ID=initiation-2
L2_CHAIN_ID=local-minitia
L1_LCD_URI=https://lcd.initiation-2.initia.xyz
L1_RPC_URI=https://rpc.initiation-2.initia.xyz
L2_LCD_URI=http://127.0.0.1:1317
L2_RPC_URI=http://127.0.0.1:26657
BRIDGE_ID=1

OUTPUT_SUBMITTER_MNEMONIC='tent apple ...'
```

| Name                        | Description                                   | Default                  |
|-----------------------------|-----------------------------------------------|--------------------------|
| L1_CHAIN_ID                 | Initia L1 Chain ID                            |                          |
| L2_CHAIN_ID                 | Your minitia's Chain ID                       |                          |
| L1_LCD_URI                  | L1 node LCD URI                               | [http://127.0.0.1:1317](http://127.0.0.1:1317)   |
| L1_RPC_URI                  | L1 node RPC URI                               | [http://127.0.0.1:26657](http://127.0.0.1:26657) |
| L2_LCD_URI                  | L2 node LCD URI                               | [http://127.0.0.1:1317](http://127.0.0.1:1317)   |
| L2_RPC_URI                  | L2 node RPC URI                               | [http://127.0.0.1:26657](http://127.0.0.1:26657) |
| BRIDGE_ID                   | Bridge ID                                     | ""                       |
| L1_CLIENT_ID                | L1 client id for Oracle (optional)            | ""                       |
| OUTPUT_SUBMITTER_MNEMONIC   | The menmonic of the output submitter account  | ""                       |

## Run the Output Submitter Bot

<CodeGroup>
```sh Node
npm run prod:output
```
```sh Docker
docker run -d -p 5432:5432 --network host ghcr.io/initia-labs/opinit-bots:v0.1.16 prod:output --env-file .env.output
```
</CodeGroup>
