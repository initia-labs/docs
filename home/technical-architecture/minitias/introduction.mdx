Minitias are a scalable rollup framework that leverage leverages the Initia L1 for security, liquidity, interoperability, and incentives.

## Features

### Virtual Machine Support

The Minitia framework is designed ti be able to work with multiple virtual machines (VMs). Currently, the following VMs are supported:

- EVM ([MiniEVM](/home/technical-architecture/minitias/vms/minievm/introduction))
- Move ([MiniMove](/home/technical-architecture/minitias/vms/minimove/introduction))
- Wasm ([MiniWasm](/home/technical-architecture/minitias/vms/miniwasm/introduction))
