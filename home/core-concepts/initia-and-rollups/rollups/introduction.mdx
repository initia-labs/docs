Rollups are a scalable rollup framework that leverage leverages the Initia L1 for security, liquidity, interoperability, and incentives.

## Features

### Virtual Machine Support

The rollup framework is designed ti be able to work with multiple virtual machines (VMs). Currently, the following VMs are supported:

- EVM ([MiniEVM](/home/core-concepts/initia-and-rollups/rollups/vms/minievm/introduction))
- Move ([MiniMove](/home/core-concepts/initia-and-rollups/rollups/vms/minimove/introduction))
- Wasm ([MiniWasm](/home/core-concepts/initia-and-rollups/rollups/vms/miniwasm/introduction))
