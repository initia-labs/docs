## Addresses

- `address: string` Current address in either Bech32 or hex format, depending on the configured `minitia` type.
- `initiaAddress: string` Bech32-formatted Initia wallet address of the connected account.
- `hexAddress: string` Hex-encoded Ethereum-compatible address of the connected account.
- `username?: string | null` Optional username linked to the account.


```tsx
export default function Home() {
  const { address, initiaAddress, hexAddress, username } = useInterwovenKit()

  return (
    <>
      <div>{address}</div>
      <div>{initiaAddress}</div>
      <div>{hexAddress}</div>
      <div>{username}</div>
    </>
  )
}
```

## UI Methods

- `openConnect(): void` Opens a drawer for connecting an external wallet.
- `openWallet(): void` Opens the main wallet drawer showing balances for the connected account.
- `openBridge(defaultValues: Partial<BridgeFormValues>): void`: Opens the bridge drawer to onboard assets. `BridgeFormValues` interface:

```tsx
interface BridgeFormValues {
  srcChainId?: string
  srcDenom?: string
  dstChainId?: string
  dstDenom?: string
  quantity?: string // Refers to the value entered by the user. For example, if you want to set 1 INIT as the initial bridge value, use "1" instead of "1000000".
}
```

Example: 

```tsx
export default function Home() {
  const { openConnect, openWallet, openBridge } = useInterwovenKit()

  return (
    <>
      <button onClick={openConnect}>Connect</button>
      <button onClick={openWallet}>Wallet</button>
      <button onClick={() => openBridge({ srcChainId: 'chain-id', srcDenom: 'denom', dstChainId: 'chain-id', dstDenom: 'denom', quantity: '100' })}>
        Bridge
      </button>
    </>
  )
}
```

## Transactions

Helpers for estimating and sending transactions on a rollup.

- `estimateGas` Estimates gas needed for a transaction.
- `requestTxBlock` Signs, broadcasts, and waits for block inclusion, returning the full response.
- `requestTxSync` Signs and broadcasts, returning the transaction hash immediately.
- `waitForTxConfirmation` Polls for transaction confirmation on-chain.

```tsx
import type { EncodeObject } from '@cosmjs/proto-signing'
import type { DeliverTxResponse, IndexedTx } from '@cosmjs/stargate'

interface TxRequest {
  messages: EncodeObject[]
  memo?: string
  chainId?: string       // defaults to provider's `defaultChainId`
  gasAdjustment?: number // multiplier for gas estimate (default: 1.4)
  gas?: number           // if set, uses this value instead of estimate
  fee?: StdFee | null    // if set, skips fee denom selection UI
}

interface TxQuery {
  txHash: string           // hash of the transaction to track
  chainId?: string         // chain ID where the tx was broadcast
  timeoutSeconds?: number  // fail if not found within (default: 30)
  intervalSeconds?: number // polling interval in seconds (default: 1)
}

declare function estimateGas(txRequest: TxRequest): Promise<number>
declare function requestTxBlock(txRequest: TxRequest): Promise<DeliverTxResponse>
declare function requestTxSync(txRequest: TxRequest): Promise<string>
declare function waitForTxConfirmation(params: TxQuery): Promise<IndexedTx>
```
