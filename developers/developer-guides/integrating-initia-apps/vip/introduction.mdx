This page describes the VIP integration process for Interwoven Rollups on Initia, including whitelisting, scoring, score updates, contract addresses, and stage query endpoints.

- [Whitelisting Rollup for VIP](/developers/developer-guides/integrating-initia-apps/vip/whitelisting-rollup-for-vip): 
  This is the prerequisite for joining VIP. It includes steps to add your rollup to the Initia Registry, deploy the VIP scoring contract, submit a whitelisting VIP proposal for rollup.
- [Updating VIP score](/developers/developer-guides/integrating-initia-apps/vip/updating-vip-score): 
  This section explains how to update the VIP score for users, including preparing a new stage, updating scores, and finalizing the stage.
- [Updating VIP Scoring Policy](/developers/developer-guides/integrating-initia-apps/vip/updating-vip-scoring-policy): 
  This section describes how to modify the scoring policy for VIP, including adjusting criteria and thresholds.
- [Updating Operator](/developers/developer-guides/integrating-initia-apps/vip/updating-operator): 
  This section outlines the process for changing the [operator commission](/home/core-concepts/vip/architecture#distribution-to-rollup-operators) or address.

If you are looking for a step-by-step guide, check out the following resource:

- [Integration Tutorial](/developers/developer-guides/integrating-initia-apps/vip/integration-vip)
